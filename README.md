## 📚 Conteúdo

- Rotinas de extração de dados (arquivos, web, APIs, etc.)
- Limpeza e tratamento de dados ausentes
- Detecção de outliers
- Transformações e padronização/normalização
- Engenharia de atributos (feature engineering)
- Pipelines de preparação de dados para análise/modelagem

## 🛠️ Tecnologias

- NumPy / Pandas
- Matplotlib / Seaborn (quando necessário)
- Jupyter Notebook
